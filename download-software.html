Virtual WiFi Router

Virtual WiFi Router is a free program that can be used to create Wi-Fi access points that share an Ethernet, 3G, 4G, 5G or wireless connection of a computer to multiple computers and portable devices with Wi-Fi capability. Also, can be used to create a local network between computers, so you will be able to transfer files or to share the resources of a computer (e.g. you can run a multi player game in the local WiFi network).
