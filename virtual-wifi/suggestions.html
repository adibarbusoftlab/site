Tell me how to improve Virtual WiFi

As new versions of Virtual WiFi are released, the program has on the one hand fewer bugs and on the other hand new features.

While some bugs may still exists in the program (and they will be fix eventually), when it comes to possible improvements in any ways which can be made to the Virtual WiFi Router, other than making the application available in new languages, the perspectives are quite limited. This is why I consider useful the existence of a webpage such as this one, in which the users can propose new features, that should be included in the program in the versions to come.

If you think that the program would be more useful if would have some additional functions or features, or if the program, as it is, should be changed in a particular way, you can make a suggestion about that using the comment form from below.

Each suggestion will be taken in consideration and depending on its complexity and my free time I will apply them in the newer versions of the program.
