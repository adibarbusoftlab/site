History of Virtual WiFi - What has been changed

Version 3.2.0 - 07/30/2017

New:

  • Language: Spanish

Version 3.1.9 - 25/04/2017

New:

  • Minimization to System Tray
  • Display the name of the connected devices
  • Possibility to generate a strong password for the configurated hotspot

Changed:

  • More simplified and easy to use graphical interface
  • Faster detection time for the connected devices and better performance in general

Removed:

  • Compatibility with Windows XP OS

Version 2.9.5 (portable) - 08/17/2016

  • Virtual WiFi 2.9.5 portable version was released

Version 2.9.5 - 05/22/2016

New:

  • Languages: Polish and Vietnamese

Version 2.9.4 - 04/18/2016

New:

  • Language: French
  • Compatibility extended with Windows 10

Version 2.9.3 - 04/12/2016

New:

  • Compatibility extended with Windows XP (SP3 or SP2 with WLAN driver)

Changed:

  • Appearance for some interface messages and buttons

Fixed:

  • Improved error detection when a hotspot is created and started - automatically restarting the hotspot solves some of the cases

Version 2.9.2 - 09/05/2015

New:

  • Implementation of auto refresh of the list with connected devices
  • Implementation of auto assigning IP address with registry backup

Changed:

  • Startup program procedures -> Startup time of the program was shortened

Fixed:

  • A major bug in windows 7 compatibility was removed
  • A couple of small bugs fixed

Version 2.0.1 - 08/05/2015

New:

  • Auto detection of the best Internet connection available
  • Window for showing hotspot details
  • Portable version 2.0.1 was released
  • Module for update and detect new version
  • Language: English

Changed:

  • Internal error handler -> the application became more stable
  • Organize the menus as tabs -> the application became very easy to use

Fixed:

  • Many small bugs

Version 2.0.0 Beta - 07/01/2015

Changed:

  • Static mechanism to create and manage hotspots was transformed in dynamic and asynchronous handler

Version 1.0.0 Beta - 06/01/2015

  • Implementation of basic functions
