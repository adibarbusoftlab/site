About Virtual WiFi

  • Purpose: The program transforms your computer which runs Windows OS into a virtual wireless router.
  • License: Freeware
  • Compatibility: Windows Vista, 7, 8, 8.1, 10 (32/64 bit)
  • Requirements: wireless adapter, system administrator privileges
  • Languages: English, Romanian, French, Polish, Vietnamese
