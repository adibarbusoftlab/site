Virtual WiFi (v2.9.5)

Virtual WiFi is a free program that enables software facilities to create Wi-Fi networks using only the Wireless adapter card of the computer, thus eliminating the need to use a physical router.

The resulting Wi-Fi network (hotspot) can be either local (Ad-Hoc), that can be used to create a connection between multiple computers (for example to transfer files from one computer to another or to use a computer as a server - for sharing a service, program or hardware, such as a printer) or a network that share an Internet connection existing on the computer with other devices of various types (smartphones, tablets, smart tv and others).
